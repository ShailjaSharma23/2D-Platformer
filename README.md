# 2D Platformer Game

A pixel-art 2D platformer game being developed in Unity using C#.

## Features

* Player movement
* Jumping
* Wall jumping
* Smooth camera follow
* Pixel-art platformer mechanics
* Fireball shooting system

## Tech Stack

* Unity
* C#
* Visual Studio Code
* Git & GitHub

## Current Status

Project is currently in development.

## Controls

* **A / D** → Move
* **Space** → Jump
* **Left Click** → Shoot fireball

## Planned Features

* Enemies
* Traps
* Health system
* Three-heart UI
* Attack system improvements
* Level checkpoints
* Respawn system
* Game over screen
* Pause menu
* Audio effects
* Improved animations and visual effects

The goal of this project is to learn and improve game development skills using Unity, including:

* Player mechanics
* Combat systems
